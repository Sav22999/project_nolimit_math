NoLimit Math
NoLimit Math è un software open-source e free, sviluppato principalmente da Saverio Morelli, ma anche da tutti coloro che collaborano con egli.
Il linguaggio di programmazione utilizzato è il Python (versione 3.6) con GUI PyQt5.

NoLimit Math è presente anche su PyPi ed è possibile installare la libreria semplicemente eseguendo il comando "pip install nolimit".

Il software permette di calcolare e di generare il grafico di un limite "semplice", ovvero di una funzione "(ax2+bx+c)/(ax2+bx+c)" oppure che si presenta come forma indeterminata "∞/∞".
Se, invece, si utilizza la versione Sympy si può eseguire qualsiasi funzione.
Per maggiori informazioni leggere il README presente in GitHub.
